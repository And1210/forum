Welcome to the Engineering Society Forum!

The engineering forum is a place where Queen's Engineering students can come to have their questions answered. It is pretty simple to use. If you have a question, then post it. If you see questions that you know the answer to, then answer them!

Upvote/Downvote

You are able to upvote and downvote questions and answers. It is good practive to upvote a questions that are posted and that you would also like to have answered. It is also good practive to upvote answers that you agree with so they can be seen as the "best" answer.

Subscribe to Questions

You are also able to subcribe to questions that you would like to follow. If you are subscribed to a question and someone posts an answer to the question, you will receive an email notification.

Forum Organization

The forum organizes questions through tags. Every question has to have a course code tag associated with it and additional tags can be applied if desired. For instance, MECH 111 (Physics) should have the tag "MECH 111" associated with it and you can also add a "Physics" and "Newton" tag if you would like. You can then search the forum for questions that fall under a particular class or topic.

Class Information and Ratings

Not only are you able to ask and answer questions, but you are also able to view a list of Queen's Engineering courses and rate them. The class pages include information such as the course description, professor, and credit weighting. You are also able to rate Queen's Engineering courses. Ratings occur on a 5 star scale and the average rating for each course is calculated.
